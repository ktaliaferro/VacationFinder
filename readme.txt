Vacation Finder
by Kenny Taliaferro

This web app recommends vacation destinations based on the user's interests and can be found at http://www.vacationfinder.xyz.

Files in the data_collection folder are used to gather Twitter data that the application uses and store it in a MySQL database.

Files in the webapp folder are used to run the application.  It uses data stored in a MySQL database to make recommendations.

The app uses Python, MySQL, and Flask.
